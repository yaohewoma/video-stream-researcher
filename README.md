## 📥 下载与使用

### 系统要求

- Windows 10/11 64位系统
- 无需安装 .NET 运行库（已内置）

### 使用方法

1. 下载 `video-stream-researcher.exe`
2. 双击运行程序
3. 阅读并同意免责声明
4. 输入视频URL开始解析

## ✨ 功能特性

- 🔍 支持多种视频流格式的解析和下载
- 🎬 支持视频预览和片段选择
- 🔐 支持B站登录获取高清视频
- 🔧 合并模式（FFmpegX/Mp4Merger√）
