## 📥 Загрузка и Установка

### Системные Требования

- Windows 10/11 64-bit
- .NET runtime не требуется (автономный)

### Быстрый Старт

1. Скачайте `video-stream-researcher.exe` из [Releases](../../releases)
2. Дважды щелкните для запуска
3. Прочитайте и примите отказ от ответственности
4. Введите URL видео для начала разбора

---

## 🚀 Использование

### Базовая Загрузка

1. Вставьте URL видео в поле ввода
2. Выберите путь сохранения (по умолчанию Рабочий стол)
3. Выберите опции загрузки:
   - Полное видео (с объединением аудио/видео)
   - Только аудио
   - Только видео
   - Без объединения (сохранить раздельные потоки)
4. Нажмите кнопку "Загрузить"

### Вход в Bilibili

1. Введите `bili` в поле URL
2. Отсканируйте QR-код приложением Bilibili
3. После входа вставьте URL видео
4. Наслаждайтесь загрузкой видео высокого качества
