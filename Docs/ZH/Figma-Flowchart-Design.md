# 视频流研究器 (Video Stream Researcher) - Figma 流程图设计规范

## 文档概述

本文档为 `video-stream-researcher` 项目提供完整的应用程序流程图设计规范。流程图涵盖应用程序的主要功能模块、用户交互流程、数据流转路径以及关键业务逻辑节点。

## 6. 详细流程图组件清单

### 6.1 主流程图组件 (Main Flow)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          主流程图组件清单                                   │
└─────────────────────────────────────────────────────────────────────────────┘

1. 开始节点
   - [启动应用] - 应用启动，进入就绪状态
   
2. 用户交互节点
   - [输入URL] - 文本输入
   - [选择保存路径] - 文件夹选择对话框
   - [选择下载选项] - 单选按钮组
   - [点击开始处理] - 按钮点击
   
3. 判断节点
   - [URL触发检测] - 检查是否"bili"触发登录
   - [文件存在检查] - 检查目标文件是否已存在
   - [确认模式检查] - 是否需要用户确认
   
4. 处理节点
   - [执行下载流程] - DownloadFlowManager
   - [解析视频信息] - VideoParser
   - [下载视频流] - DownloadManager
   - [合并音视频] - Mp4Merger
   - [保存下载记录] - DownloadIndexService
   
5. 状态节点
   - [解析中] - 视频信息解析
   - [下载中] - 文件下载中
   - [处理中] - 音视频合并中
   - [已完成] - 下载完成
   
6. 结束节点
   - [下载完成] - 正常完成
   - [下载失败] - 异常结束
   - [下载取消] - 用户取消
```

### 6.2 登录流程图组件 (Login Flow)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          登录流程图组件清单                                  │
└─────────────────────────────────────────────────────────────────────────────┘

1. 触发节点
   - [输入"bili"] - 用户输入触发
   
2. 处理节点
   - [初始化登录] - LoginService.InitializeAsync
   - [生成二维码] - LoginService.StartQrCodeLoginAsync
   - [显示二维码窗口] - QrCodeWindow
   
3. 等待节点
   - [等待扫码] - WaitingForScan
   - [已扫码待确认] - ScannedWaitingConfirm
   
4. 回调节点
   - [登录成功回调] - LoginSuccess 事件
   - [登录失败回调] - LoginFailed 事件
   - [状态变更回调] - LoginStatusChanged 事件
   
5. 结果节点
   - [登录成功] - 显示用户名
   - [登录失败] - 显示错误信息
   - [登录过期] - 提示重新登录
```
