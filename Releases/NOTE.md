# 编译说明

## 当前状态

由于项目结构调整，目前通过命令行编译存在类型冲突问题。

## 推荐编译方式

### 方法 1: 使用 Visual Studio 2022（推荐）

1. 打开 `video-stream-researcher.sln` 解决方案文件
2. 选择 `Release` 配置
3. 右键项目 → 发布
4. 选择 `win-x64` 目标运行时
5. 勾选 `自包含` 和 `单文件`
6. 发布到 Releases 文件夹
