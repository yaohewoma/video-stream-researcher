# Figma 流程图设计系统

## 设计原则

- **简洁现代**: 使用圆角、阴影、渐变营造现代感
- **层次分明**: 通过颜色、大小、间距区分层级
- **视觉引导**: 使用箭头、连线颜色引导阅读路径
- **一致统一**: 所有元素遵循统一的设计规范

## 9. 流程图绘制步骤

### 9.1 准备工作

1. 创建新 Figma 文件
2. 设置页面尺寸: 1920 × 1080
3. 导入设计系统颜色
4. 创建字体样式
5. 导入图标库

### 9.2 绘制流程

1. **规划布局**
   - 确定流程方向
   - 划分功能区域
   - 预留注释空间

2. **放置节点**
   - 从主要流程开始
   - 使用 8px 网格对齐
   - 保持统一间距

3. **连接节点**
   - 使用直线或折线
   - 添加流向标签
   - 区分主次路径

4. **添加样式**
   - 应用颜色编码
   - 添加阴影效果
   - 统一字体样式

5. **添加注释**
   - 功能说明
   - 数据标注
   - 注意事项
