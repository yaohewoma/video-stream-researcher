## 核心流程图解

### 下载视频流程

```
开始
  ↓
输入视频URL
  ↓
是B站视频? ──是──→ 已登录? ──否──→ 提示登录 ──→ 返回输入
  │              │
  否            是
  │              │
  └──────────────┘
                 ↓
           文件已存在?
              │
        是 ───┴─── 否
        │          │
   提示已存在   解析视频信息
        │          │
        └────┬─────┘
             ↓
       选择下载模式
    (完整/仅音频/仅视频)
             │
             ↓
       下载视频流
             │
             ↓
       需要合并? ──是──→ 合并音视频
             │              │
            否              │
             └──────────────┘
                            ↓
                      保存到本地
                            ↓
                      下载成功
                            ↓
                      结束
```

### B站登录流程

```
开始
  ↓
输入 'bili'
  ↓
已经登录? ──是──→ 自动登出
  │                  │
  否                 │
  │                  │
  └────────┬─────────┘
           ↓
    生成二维码
           ↓
    弹出二维码窗口
           ↓
    等待用户扫码
           │
    已扫码? ──否(超时/取消)──→ 登录失败
           │
          是
           ↓
    等待手机确认
           │
    确认? ──否(取消/超时)──→ 登录失败
           │
          是
           ↓
    登录成功
           ↓
    关闭窗口
           ↓
    显示用户名
           ↓
    结束
```
